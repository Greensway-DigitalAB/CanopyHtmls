v1.8

  • Du kan nu snabbt nollställa alla filter som du gjort av data i Project manager.

  • Ikonerna på projektsidan har ordnats om för att göra tydligare och du kan nu också skapa PDF-rapporter åt flera naturvärdesbedömningar med ett enkelt klick. Välj att skapa rapporter för alla NVB:er i projektet eller filtrera ut de du vill skapa rapporter för.

  • För att förbättra överblicken har vi gjort några mindre förändringar i designen. Du kan nu visa fler rader av enkäter, sortera dem enklare samt kopiera vyn, exportera vyn till MS Excel eller skriva ut den direkt.

  • Om felaktiga polygoner skapas i appen och blir så kallade multipolygons. Nu kan du ordna det i Project Manager.

  • Vi har även förbättrat din hantering av användare.

  • Det går enkelt att skicka användarens inloggningsuppgifter igen som administratör.

  • Om användaren slutar på företaget kan du nu snabbt inaktivera dennes konto.

  • Om användaren försöker logga in med fel lösenord sex gånger i rad blir dennes konto inaktiverat automatiskt för att förhindra hackning. Du kan då aktivera kontot igen från din admin-sida. Sök upp användaren i ”inaktiva användare” och klicka på aktivera konto.
